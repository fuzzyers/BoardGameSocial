# BoardGameSocial

This repository is split into two parts:

- `frontend/` — the React Native Expo app
- `backend/` — backend services and APIs

## Frontend Setup

1. Change into the frontend folder:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
4. Run on Android:
   ```bash
   npm run android
   ```
5. Run on iOS:
   ```bash
   npm run ios
   ```

## Notes

- The frontend uses Expo SDK 54.
- The backend folder is ready for your API code.
